# Rental-Price-Analysis-Data-Mining
Data Mining &amp; Machine Learning Analysis on Rental Pricing Dataset (Malaysia)

📌 Overview

This project analyzes rental property data in Kuala Lumpur and Selangor using Python to uncover key factors influencing rental prices and market trends.

🎯 Objective

Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Identify key factors affecting rental prices
Generate insights for tenants, landlords, and policymakers

🛠 Tools & Technologies

Python (Pandas, Matplotlib, Seaborn)
Jupyter Notebook (.ipynb)
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)

📊 Key Analysis Performed

Data cleaning (handling missing values, formatting)
Descriptive statistics
Data visualization (bar charts, line graphs, heatmaps)
Correlation analysis between features

🔍 Key Insights

📍 Kuala Lumpur properties generally have higher rental prices than Selangor
🏢 Condominiums dominate the rental market and have the highest rental values
🛋️ Fully furnished properties command higher rental prices
📈 Newer properties tend to have higher rental prices due to modern amenities
